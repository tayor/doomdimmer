# Security Policy

Report security issues through GitHub issues if the report is not sensitive:

```text
https://github.com/tayor/doomdimmer/issues
```

For sensitive reports, open an issue asking for a private disclosure channel and include only a brief non-sensitive summary.

## Security Expectations

- No remote code execution.
- No analytics or telemetry.
- No server-side processing.
- No collection of page content.
- No raw scroll log persistence.
- No unnecessary Chrome permissions.
- User-provided prompt text is rendered safely.
