# DoomDimmer landing page redesign

Static HTML/CSS implementation matching the provided dark purple DoomDimmer mockup.

## Files

- `index.html` — landing page
- `privacy.html` — privacy policy page using the same visual system
- `styles.css` — complete responsive styling
- `icon128.png`, `logo.png` — local logo/favicon assets

Open `index.html` directly in a browser or serve the folder with any static host.
