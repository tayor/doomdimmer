Privacy Policy for DoomDimmer

Last updated: May 16, 2026

Overview

DoomDimmer is a local-first Chrome extension that helps interrupt mindless scrolling. Detection runs in the user's browser.

Data Handling

DoomDimmer processes scroll timing, scroll density, tab visibility, focus, and engagement events locally. DoomDimmer does not sell, share, or transmit browsing activity, page content, scroll behavior, settings, or optional local stats to the developer or third parties.

Data Stored Locally

DoomDimmer may store settings, site rules, restore phrase, pause and cooldown timestamps, onboarding state, and optional aggregate local stats in chrome.storage.local.

Local stats are disabled by default, contain aggregate counts only, and can be cleared from settings. DoomDimmer does not store raw scroll logs, page text, typed page contents, or browsing history.

Network Use

No analytics. No external API calls. No server processing. No remote AI.

Permissions

DoomDimmer requests local storage and access to normal http:// and https:// sites so it can detect scrolling behavior and apply reversible visual effects on enabled pages.

Advertising and Sale of Data

DoomDimmer does not serve advertising and does not use, transfer, or sell user data for advertising, retargeting, credit decisions, or data broker services.

Limited Use Statement

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact

For support or privacy questions, use GitHub issues.
